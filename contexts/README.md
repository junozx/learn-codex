# Contexts

These files are project context presets referenced by `AGENTS.md`.

## Files

1. `dev.md`: default implementation mode.
2. `review.md`: review/audit mode.
3. `research.md`: investigation/comparison mode.

## Usage

1. If no mode is specified, use `dev.md`.
2. For code review requests, use `review.md`.
3. For research requests, use `research.md`.
