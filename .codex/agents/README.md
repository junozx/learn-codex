# Custom Subagents

Project-scoped custom subagents live in this directory.

## File Format

1. One agent per `.toml` file.
2. Required fields:
3. `name`
4. `description`
5. `developer_instructions`
6. Optional fields include `sandbox_mode`, `model`, `reasoning_effort`.
