# Project Skills

This directory contains project-scoped Codex skills.

Each skill must live under:

`.agents/skills/<skill-name>/SKILL.md`

Recommended format:

1. Frontmatter with `name` and `description`.
2. Persistent instructions.
3. Optional resources (scripts/docs/examples).
